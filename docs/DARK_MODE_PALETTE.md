# Sagrada Madre CRM - Dark Mode Design System
> Proyección de paleta de colores para la futura implementación del Modo Oscuro (Dark Mode).

## Filosofía del Diseño
El objetivo del Dark Mode para "Sagrada Madre" no es utilizar negros puros (`#000000`) o grises genéricos (`#333333`), sino mantener la temperatura cálida, premium y espiritual de la marca utilizando tonos carbón-berenjena.

## Paleta de Colores (TailwindCSS Referencia)

### 1. Fondos Generales (El Canvas)
* **Color Base:** Carbón-Berenjena Profundo
* **Hex:** `#1A1525` o `#1E1B24`
* **Uso:** Todo el fondo principal del Dashboard (`bg-slate-900` equivalente modificado).

### 2. Superficies (Tarjetas, Paneles, Kanban)
* **Color Base:** Púrpura-Grisáceo Elevado
* **Hex:** `#2A2438` o `#352F44`
* **Uso:** El fondo de las cards del Kanban, modales, y contenedores que van por encima del Canvas. Genera efecto de elevación.

### 3. Acentos Principales (Botones y Acciones)
* **Color Base:** Lavanda Luminoso
* **Hex:** `#A378D6` o `#8A5AC2`
* **Uso:** Hover state del sidebar, botones primarios (Iniciar Sesión, Añadir Trato). En fondo oscuro se requiere invertir la luminosidad para destacar.

### 4. Acento Crítico y Alertas (El Oro)
* **Color Base:** Oro Brillante Cálido
* **Hex:** `#E5C885` o `#F2CD76`
* **Uso:** Alertas, iconos destacados, estado de tratos "Ganados". Debe brillar y no verse como mancha café oscura.

### 5. Tipografías (El Texto)
* **Color Base:** Crema Suave / Hueso
* **Hex:** `#E6E1D8` o `#F4F0EB`
* **Uso:** Color principal de títulos, párrafos y nombres de clientes. Reduce la fatiga visual nocturna generada por el blanco puro `#FFFFFF`.
